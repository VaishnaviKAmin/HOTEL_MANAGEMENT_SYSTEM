Database Operations Demo

  • Stored Procedures: Calculate billing sums and query availabilities dynamically based on date intervals.

Trigger Scenario Demonstrations

Interact with real-world scenarios to see MySQL triggers dynamically run and protect data integrity.

Trigger 1

Room Status Auto-Update

When a guest makes a booking for a room, a trigger (`after_booking_insert`) automatically updates that room's status column to 'Booked'. Similarly, deleting a booking resets it to 'Available'.

Test Status Update Trigger

Trigger 2

Double Booking Prevention

A trigger (`prevent_double_booking`) intercepts insertion on the Booking table. If another booking already reserves the same room during overlapping dates, the transaction is rejected and MySQL signals an error.
